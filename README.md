# Mental Health Backend

This is the backend codebase for a mental health app focusing on combating rumination, setting goals, and expressing gratitude.

## Features
- User authentication
- Journaling
- Goal tracking
- Gratitude entries
- AI integration for sentiment analysis (in progress)

## Tech Stack
- Django
- PostgreSQL
- Supabase
